Waterfall

Waterfall is a simple CSS layout grid based on fractions.

Waterfall us designed for desktop and mobile and is compatible with Internet Explorer 9 and above.

Waterfall is not standards compliant, because it uses custom HTML tags and attribute.

Waterfall is at version v0.0.
